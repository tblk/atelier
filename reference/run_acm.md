# Exécuter une ACM

Exécuter une ACM

## Usage

``` r
run_acm(data_prepare, excl = NULL, quali_sup = NULL, ncp = 6)
```

## Arguments

- data_prepare:

  données préparées

- excl:

  fonction ou vecteur de modalités à exclure

- quali_sup:

  indices des variables supplémentaires

- ncp:

  nombre de dimensions

## Value

objet FactoMineR::MCA
