# Construire un tableau consolidé des résultats d'ACM

Construire un tableau consolidé des résultats d'ACM

## Usage

``` r
tidy_acm(res_acm)
```

## Arguments

- res_acm:

  Objet résultat d'une ACM (ex: FactoMineR::MCA)

## Value

Un data.frame contenant les modalités actives, exclues et
supplémentaires.
