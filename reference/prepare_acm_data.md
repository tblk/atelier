# Préparer des données pour une ACM

Préparer des données pour une ACM

## Usage

``` r
prepare_acm_data(data, var_actives, var_sup = NULL)
```

## Arguments

- data:

  data.frame source

- var_actives:

  variables actives

- var_sup:

  variables supplémentaires

## Value

data.frame prêt pour FactoMineR::MCA
