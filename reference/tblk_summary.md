# Personnalised tbl_summary

Personnalised tbl_summary

## Usage

``` r
tblk_summary(data, vars, by = NULL, missing_text = "Manquant")
```

## Arguments

- data:

  A dataframe

- vars:

  A named list, with variable names as names, and labels

- by:

  Optionnal grouping variable

- missing_text:

  Label for missing text

## Value

A gtsummary::tbl_summary()
